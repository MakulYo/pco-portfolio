First feedback

Posted by Mikołaj • 2025-09-11

Early feedback focused on structure and clarity: make the navigation consistent, add alt text, and keep the layout responsive. I also learned how one missing character can break a page (a meta tag that swallowed my CSS link).

I fixed those by standardising the header, adding a skip link, and using a three-stage layout for the aside (stacked on mobile, sticky on tablet, fixed on desktop). I also added clear titles and descriptions per page.

Next steps: keep content short and specific, and improve the dashboard table with EC values and a clear NBSA marker.
